# Portfólio Profissional — João Vitor Costa

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido com o objetivo de apresentar meus projetos, habilidades e trajetórias na área de desenvolvimento de software.

---

## 💻 Sobre o Projeto

O projeto consiste em uma aplicação web de página única (Single Page Application) focada na experiência do usuário, integrando interfaces responsivas, animações controladas por rolagem e um canal de comunicação direta.

### Principais Recursos

- Apresentação Profissional: Seções dedicadas ao histórico e objetivos profissionais.
- Galeria de Projetos: Exibição horizontal automatizada de 12 projetos.
- Efeitos Visuais Avançados: Uso de profundidade 3D interativa (tilt) e desfoque de fundo (backdrop filter).
- Formulário de Contato: Integração com a API do WhatsApp para envio automatizado de mensagens.
- Responsividade: Menu de navegação adaptado para dispositivos móveis (mobile-first).

---

## 🛠️ Tecnologias Utilizadas

- HTML5 e CSS3: Estruturação semântica e estilização moderna com uso de variáveis CSS e animações nativas.
- JavaScript (ES6+): Lógica de controle do menu responsivo e manipulação do DOM.
- GSAP & ScrollTrigger: Biblioteca utilizada para criar a linha de tempo e gerenciar a transição do scroll vertical para horizontal na seção de projetos.
- Lenis Scroll: Implementação de rolagem suave (smooth scroll) integrada ao motor de animação.
- VanillaTilt.js: Biblioteca responsável pelo efeito reativo de inclinação nos cartões de projetos.

---

## 📂 Estrutura do Site

1. Início: Introdução com foto de perfil, nome e atribuição profissional (Desenvolvedor Front End e Programador de Sistemas).
2. Sobre Mim: Seção textual reservada para o perfil profissional.
3. Meus Projetos: Área com fixação de tela (pin) onde os cartões de projetos transitam horizontalmente conforme a rolagem da página.
4. Entre em Contato: Formulário estruturado para coleta de nome e mensagem.
5. Rodapé: Direitos autorais e encerramento com plano de fundo em vídeo.
