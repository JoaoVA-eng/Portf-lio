Sobre Mim

Desenvolvedor de Software focado em criar soluções web eficientes, dinâmicas e visualmente impactantes. Tenho experiência no desenvolvimento de interfaces responsivas utilizando HTML5, CSS3 e JavaScript moderno (ES6+), além da integração de bibliotecas de animação e performance como GSAP e ScrollTrigger para otimizar a experiência do usuário. Busco constantemente aplicar as melhores práticas de clean code, semântica e performance para entregar projetos robustos que resolvam problemas reais.
